# Wardrobe Log — Support & Privacy

Static support and privacy pages for the **Wardrobe Log** iOS / macOS app,
hosted with GitHub Pages.

## Pages

- `index.html` — landing page
- `support.html` — support / FAQ
- `privacy.html` — privacy policy

## URLs for App Store Connect

Once GitHub Pages is enabled (see below):

- **Support URL:** `https://camipera.github.io/wardrobe-log-support/support.html`
- **Privacy Policy URL:** `https://camipera.github.io/wardrobe-log-support/privacy.html`

## Enabling GitHub Pages (one time)

1. Open this repo on GitHub → **Settings** → **Pages**.
2. Under **Build and deployment**, set **Source** to **Deploy from a branch**.
3. Choose branch **`main`** and folder **`/ (root)`**, then **Save**.
4. Wait ~1 minute, then confirm the pages load at the URLs above.
